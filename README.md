## Neovim

### Markdown Preview
Sometimes, the installation can be problematic. If you run into issues, try the following steps:
1. Ensure you have Node.js installed on your system. You can download it from [nodejs.org](https://nodejs.org/).
2. Open your terminal and run the following command to install the `markdown-preview.nvim` plugin

    ```bash
    :CocInstall coc-markdown-preview-enhanced
    ```

3. Or manually install the plugin with npm:

    ```bash
    cd ~/.local/share/nvim/lazy/markdown-preview.nvim/app
    npm install
    ```
